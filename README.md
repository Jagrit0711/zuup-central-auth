# Zuup Auth

Centralized authentication system for all Zuup services.

## Features

- Single Sign-On (SSO)
- OAuth 2.1 & OIDC support
- User profile management
- App registration and management

## Tech Stack

- React + TypeScript + Vite
- shadcn/ui + Tailwind CSS
- Supabase Auth
